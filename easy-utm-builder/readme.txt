Easy to build trackable URLs with UTM parameters in Bulk (complete site or specific post type) for Google Analytics!

== Description ==


This plugin will add a page called "Easy UTM Builder" under Settings. On this screen, you need to select post type, set the UTM parameter and select output type.

= Main Features: =
* Easy to use
* Easy to generate
* Generate for complete/partial site
* Output in Dashboard
* Write in CSV file


You can create URLs manually but what if you have Hundreds/Thousands URLs. In that case, it'll Cost you a lot of Time. We all know, Time is Money and you can save it by using this Free Plugin!

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Easy UTM Builder'
3. Click on 'Install Now'
4. Activate Easy UTM Builder from your Plugins page.

= From WordPress.org =

1. Download Easy UTM Builder.
2. Unzip plugin.
2. Upload the 'Easy UTM Builder' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Easy UTM Builder from your Plugins page.

= Usage =

1. Go to Settings > Easy UTM Builder to build UTM URLs.
2. Select Post Type
3. Enter Source
4. Enter Medium
5. Enter Name
6. Enter Term (optional)
7. Enter Content (optional)
8. Select Output Type
9. Click on "Build UTM URLs"


= Uninstalling: =

1. In the Admin Panel, go to "Plugins" and deactivate the plugin.
2. Delete the plugin from Plugins page.
